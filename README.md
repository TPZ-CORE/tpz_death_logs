
# TPZ-CORE Crafting

## Requirements

1. TPZ-Core: https://github.com/TPZ-CORE/tpz_core
2. TPZ-Characters: https://github.com/TPZ-CORE/tpz_characters
3. TPZ-Inventory : https://github.com/TPZ-CORE/tpz_inventory
4. TPZ-Metabolism: https://github.com/TPZ-CORE/tpz_metabolism
5. TPZ-Medics: https://github.com/TPZ-CORE/tpz_medics

# Installation

1. When opening the zip file, open `tpz_death_logs-main` directory folder and inside there will be another directory folder which is called as `tpz_death_logs`, this directory folder is the one that should be exported to your resources (The folder which contains `fxmanifest.lua`).

2. Add `ensure tpz_death_logs` after the **REQUIREMENTS** in the resources.cfg or server.cfg, depends where your scripts are located.
